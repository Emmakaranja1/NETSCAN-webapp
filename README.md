## Project Overview
**NetScan** is a professional-grade network diagnostic tool designed for Data Centre Operations (DCO) environments. It provides real-time IP/Domain intelligence, infrastructure mapping, and system logging through a high-performance "Cyber/Terminal" interface.

This project was developed to demonstrate:
- **Automation & Scripting**: Real-time API integration and state management.
- **Networking Fundamentals**: Deep understanding of TCP/IP, DNS, ISP routing, and ASN mapping.
- **Infrastructure Monitoring**: Dashboard design focused on operational visibility.
- **Cybersecurity**: Information gathering and target reconnaissance.

## Key Features
- **Terminal UI**: A monospace, command-line inspired interface that mimics enterprise Linux environments.
- **Real-time Diagnostics**: Direct integration with `ipwhois.app` for instant target resolution.
- **Infrastructure Dashboard**: Detailed breakdown of ISP, ASN, and Geographic data.
- **System Logs**: Live-scrolling operational logs to track scan history and system status.
- **Responsive Design**: Fully optimized for both desktop "Mission Control" views and mobile field operations.

## Tech Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS 4.
- **Animations**: Motion (formerly Framer Motion) for smooth UI transitions.
- **Icons**: Lucide React for consistent technical iconography.
- **API**: RESTful integration with IP Geolocation services.

## Relevance to iXAfrica Data Centre Role
1. **Linux CLI Focus**: The UI is built to feel native to a Linux administrator, emphasizing the importance of terminal-based workflows.
2. **Networking Knowledge**: Demonstrates practical application of DNS resolution and IP routing concepts.
3. **Monitoring & Visibility**: Showcases the ability to build tools that provide clear, actionable infrastructure data—critical for managing a Tier 3 data centre like NBOX1.

## How to Use
1. **Acquisition**: Enter any IP address or Domain name (e.g., `ixafrica.co.ke`) into the target acquisition module.
2. **Execution**: Click the scan button or press Enter.
3. **Analysis**: Review the resolved infrastructure data in the dashboard and monitor the system logs for real-time status updates.
